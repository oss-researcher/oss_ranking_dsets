# Replication Package

This repository contains datasets, code, and supporting materials used in the study.

## Contents
- data/: raw and processed datasets
- code/: scripts to reproduce results
- excel/: supporting calculations
- output/: figures and tables

## Reproducibility
1. Run preprocessing.R  
2. Run analysis.R  
3. Outputs will be generated in /output  

All identifying information has been removed for double-blind review.
